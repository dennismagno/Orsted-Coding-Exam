### Installation

1. Clone the repo:

   git clone https://github.com/dennismagno/orsted-coding-exam

2. Change directory:

   cd orsted-coding-exam/employee-app

3. Restore packages:

   dotnet restore
   
4. Run application:

   dotnet run
